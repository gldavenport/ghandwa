# §3.2.5 Worked Examples

The following derivations trace a PIE input through the complete rule sequence. Each illustrates a distinct set of rule interactions. All forms are drawn from attested lexicon entries.

## §3.2.5.1 \*wirós ~ \*wirósyo 'man' (o-stem, masc.)

| Step | Nominative | Genitive |
|---|---|---|
| Input | wiros | wirosyo |
| y → j (§3.2.3.5) | — | wirosjo |
| s → z (§3.2.3.4) | — | wirozjo |
| **Output** | ***wiros*** | ***wirozio*** |

A transparent derivation: no laryngeals, no syllabic resonants, no clusters. The only changes are Stage 2 voicing of intervocalic *s* and the orthographic *y* → *j*.

## §3.2.5.2 \*h₂ŕ̥tḱos 'bear' (o-stem, masc.)

| Step | Form |
|---|---|
| Input | h₂r̥tḱos |
| Centumization (§3.2.1.1) | h₂r̥tkos |
| H-initial deletion (§3.2.2.3d) | r̥tkos |
| Thorn metathesis (§3.2.2.2) | r̥ktos |
| Syllabic resonant (§3.2.3.1) | arktos |
| **Output** | ***arktos*** |

Centumization feeds thorn metathesis by converting ḱ to k, creating the TK sequence that undergoes metathesis. The initial laryngeal is lost without trace (no preceding vowel to color or lengthen). The syllabic resonant vocalizes in Stage 2 to give the characteristic Ghandwa *ar*.

## §3.2.5.3 \*ph₂tḗr 'father' (r-stem, masc.)

| Step | Form |
|---|---|
| Input | ph₂tēr |
| CHC → CaC (§3.2.2.3e) | patēr |
| **Output** | ***patēr*** |

The laryngeal h₂ is caught between two consonants (p and t) in the initial syllable and resolves to *a*. No further rules apply. Cf. PGmc \*fadēr < \*pətēr, with the same epenthetic vowel.

## §3.2.5.4 \*h₁nḗh₃mn̥ 'name' (men-stem, neut.)

| Step | Form |
|---|---|
| Input | h₁nēh₃mn̥ |
| H-initial deletion (§3.2.2.3d) | nēh₃mn̥ |
| Laryngeal coloring (§3.2.2.3a) | nōmn̥ (h₃ colors ē → ō) |
| VH → V̄ (§3.2.2.3c) | nōmn̥ (ō already long; h₃ deleted) |
| Osthoff's Law (§3.2.2.5) | nomn̥ (ō shortens before m + C) |
| Syllabic resonant (§3.2.3.1) | noman |
| **Output** | ***noman*** |

This form demonstrates the interaction of laryngeal coloring, compensatory lengthening, and Osthoff's Law. The h₃ first colors the adjacent vowel (ē → ō), then is deleted with compensatory lengthening (but the vowel is already long, so the length is redundant). Osthoff's Law then shortens the long ō before the cluster mn̥. Finally, the syllabic nasal vocalizes in Stage 2.

## §3.2.5.5 \*dʰugh₂tḗr 'daughter' (r-stem, fem.)

| Step | Form |
|---|---|
| Input | dʰugh₂tēr |
| CHC → CC (§3.2.2.3e) | dʰugtēr (h₂ between g and t, post-vocalic → ∅) |
| Obstruent devoicing (§3.2.2.4) | dʰuktēr (g → k before t) |
| Aspirate fricativization (§3.2.3.3) | ðuktēr |
| **Output** | ***ðuktēr*** |

The ordering of devoicing before fricativization is critical here. The medial *g* must devoice to *k* before *t* (§3.2.2.4) while it is still a stop. Then the initial *dʰ* fricativizes to *ð* (§3.2.3.3). If fricativization applied first, *g* would remain a voiced stop before *t*, incorrectly surviving as *ɣ* through the wrong ordering.

## §3.2.5.6 \*h₂yúHn̥ḱos 'young' (o-stem, masc.)

| Step | Form |
|---|---|
| Input | h₂yúHn̥ḱos |
| Centumization (§3.2.1.1) | h₂yuHn̥kos |
| H-initial deletion (§3.2.2.3d) | yuHn̥kos |
| VH → V̄ (§3.2.2.3c) | yūn̥kos |
| Osthoff's Law (§3.2.2.5) | yun̥kos (ū before n̥ + k → u) |
| Syllabic resonant (§3.2.3.1) | yuankos |
| Hiatus glide insertion (§3.2.3.2) | yuwankos (u before a → u.wa) |
| y → j (§3.2.3.5) | juwankos |
| **Output** | ***juwankos*** |

The *uw* sequence arises from hiatus glide insertion (§3.2.3.2), not from the *ū* itself. After Osthoff's Law shortens *ū* to *u*, syllabic resonant vocalization produces the sequence *u.a* (from *un̥* → *uan*), which creates a hiatus environment. The glide insertion rule then resolves it: *u.a* → *u.wa*. The derivation thus passes through three distinct stages: long *ū* (from compensatory lengthening), short *u* (from Osthoff), and *uw* (from hiatus resolution).

## §3.2.5.7 \*kʷekʷlos 'wheel' (o-stem, masc.)

| Step | Form |
|---|---|
| Input | kʷekʷlos |
| Ghandwa Boukólos (§3.2.1.4) | kʷeklos (second kʷ delabializes before l) |
| **Output** | ***kʷeklos*** |

The second labiovelar stands before the liquid *l* — a consonant — and delabializes by the Ghandwa Boukólos rule (§3.2.1.4). The first labiovelar, before the vowel *e*, is preserved. This is a minimal illustration of labiovelar distribution in Ghandwa: preserved before vowels, simplified before consonants.

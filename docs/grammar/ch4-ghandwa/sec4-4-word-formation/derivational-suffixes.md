## *-el-* Diminutive

**Source:** Inherited directly from PIE \*-elo-. Formally identical with the PIE suffix; no Ghandwa innovation.

**Comparanda:** Greek *-ελος* (e.g. diminutive and hypocoristic formations); Latin *-ulus* < \*-olo-* — a divergent Italic reflex showing vowel lowering *e→o* followed by rhotacism in some environments. The Latin reflex confirms the inherited status of the suffix while showing that Ghandwa *-el-* is not an independent innovation.

**Thematic forms:**

| Form | Gender/class | Notes |
|---|---|---|
| *-elos* | animate/common (thematic masc.) | |
| *-elom* | neuter (thematic neut.) | |
| *-elā* | feminine (ā-stem) | |

**Attachment:** The suffix attaches directly to the nominal stem. No intervening vowel.

**Productivity:** Regular and productive.

**Examples:**

| Base | Diminutive | Meaning |
|---|---|---|
| *ekʷos* 'horse' | *ekʷelos* | little horse, foal |
| *morjekʷos* 'sea-horse' | *morjekʷelos* | little sea-horse, seahorse (fish) |
| *akʷekʷos* 'river-horse' | *akʷekʷelos* | little river-horse, young hippopotamus |

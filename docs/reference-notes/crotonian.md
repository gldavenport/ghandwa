# Crotonian+ Sound Changes

## Stage 0 — inherited PIE / pre-Crotonian

### 0.1 Centum merger

- *Ḱ > K*

### 0.2 Adjacent laryngeal coloring

- *h₁e, eh₁ > he, eh*
- *h₂e, eh₂ > xa, ax*
- *h₃e, eh₃ > xo, ox*

---

## Stage 1 — core Crotonian phonology

### 1.1 PIE voiced aspirates become plain voiceless stops

- *Dʰ > T*

Specifically:

- *bʰ > p*
- *dʰ > t*
- *gʰ > k*
- *gʷʰ > kʷ*

### 1.2 Vowel merger

- *o > a*

---

### 4.1 Monophthongization

# Overall Profile

Crotonian+ is:

- centum
- heavily *a*-colored
- preserves fossilized internal laryngeal traces
- deletes initial laryngeals before consonants
- treats syllabic resonants aggressively
- preserves Garnier’s diagnostic *Dʰ > T*
- archaic and substrate-like without being explicitly Anatolian or Lydian
- intentionally “strange but still Indo-European”

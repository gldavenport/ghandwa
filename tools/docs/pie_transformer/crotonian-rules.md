# Crotonian and wékʷos Rule Sets

## 1. wékʷos Rule Set

Notation: V = any vowel; P = any stop; R = non-syllabic resonant (*r, l, m, n*); L = liquid (*r, l*); Lab = labial sonorant (*m, w*); H = any laryngeal; Dʰ = any voiced aspirate (*bʰ, dʰ, gʰ, gʷʰ*); T = voiceless plain stop; D = voiced plain stop; Kʷ = labiovelar stop series (*kʷ, gʷ*); DENT = dental stop {t, d} (*s* and *n* excluded); x = velar fricative (h₂/h₃ residue, after 2.7 merger); xʷ = labiovelar fricative (h₃ residue, before 2.7 merger); `.` = syllable boundary.

| # | Rule | Notes |
|---|---|---|
| 1.1 | Ḱ > K | Centum; palatovelar merges with plain velar |
| 2.1 | #HC- > #C- | Initial laryngeal before consonant deleted; no trace |
| 2.2 | CH# > C# | Word-final laryngeal after consonant deleted |
| 2.3 | CHC > CaC | Medial laryngeal between consonants vocalizes as *a*; ordered before 2.5 while H is still H |
| 2.4 | CVRH.C > CVRa.C | Laryngeal after resonant in post-vowel cluster vocalizes as *a*; specific case of 2.3 and takes priority when both could apply |
| 2.5 | h₁ > ∅; h₂ > x; h₃ > xʷ; Vh₁ > V̄ | Remaining h₁ deleted; h₂/h₃ consonantalize; post-vocalic h₁ lengthens preceding vowel |
| 2.6 | xe, ex > xa, ax; xʷe, exʷ > xʷo, oxʷ | Coloring: x colors adjacent *e* > *a*; xʷ colors adjacent *e* > *o* |
| 2.7 | xʷ > x | h₃-residue merges with h₂-residue after coloring is complete |
| 3.1 | DʰL > DL | Aspirate before liquid loses aspiration; bleeds 3.2 for DʰL inputs; DL output is final — no subsequent rule alters D in this environment |
| 3.2 | Dʰ > T | Remaining voiced aspirates devoice: bʰ > p, dʰ > t, gʰ > k, gʷʰ > kʷ; note gʷʰ > kʷ enters the 11.1–11.2 chain at word-final position |
| 3.3 | DENT₁DENT₂ > DENT₂DENT₂ | Dental cluster assimilation |
| 4.1 | CuR̥C > CuRC; CiR̥C > CiRC | Glide-resonant resyllabification before general vocalization; pre-vocalic *u*, *i* normalized from glide notation |
| 4.2 | #(s)kRV > #(s)kR̥V | Initial cluster syllabification; resonant made syllabic, feeds 4.3 |
| 4.3 | m̥, n̥, r̥, l̥ > am, an, ar, al | Syllabic resonant vocalization |
| 4.4 | CVLC > CLVC | Liquid metathesis; L = {r, l} only; nasals do not metathesize |
| 5.1 | VTNV > VNDV | Nasal anticipation; voiceless stop before nasal voices to D; aspirates already resolved by 3.2 so D input is vacuous |
| 5.2 | #pr > #br | Initial labial+liquid voices; dental-initial clusters (*#tr*) have no reflex distinct from input — no fortis distinction in wékʷos |
| 6.1 | Barytonesis | Stress retraction to initial syllable; feeds 10.1 |
| 7.1 | #xLV > #kLV | Initial laryngeal residue fortifies before liquid+vowel onset |
| 7.2 | Vx > V̄ | Post-vocalic laryngeal residue gives compensatory lengthening |
| 7.3 | x > ∅ | Remaining laryngeal residue deleted |
| 8.1 | #LabLV > #bLV | Initial labial sonorant+liquid+vowel voices to *b*; Lab = {m, w} |
| 9.1 | ej > ī; ow > ū | Collapse of homorganic diphthongs |
| 9.2 | ew > ow; oj > ej | Harmonization of heterorganic diphthongs; 9.1 precedes 9.2 — products of 9.2 are post-9.1 and not subject to it |
| 10.1 | V̄ > V / _# (polysyllables) | Long vowels shorten in final unstressed syllable; monosyllables exempt |
| 10.2 | o > a | Short *o* merges with *a*; long *ō* unaffected |
| 10.3 | s > z / {V,R}_{V,R} | Intervocalic *s* voices between vowels and resonants; ordered after 10.2 so merged *a* environments are included |
| 11.1 | p, t, k, kʷ, s > b, d, g, gʷ, z / _# | Word-final obstruent voicing |
| 11.2 | Kʷ > K | Labiovelar delabialization; full series: kʷ > k, gʷ > g; ordered after 11.1 so final kʷ > gʷ > g |

## 2. Dropped rule: Verner-like alternation

Garnier & Sagot's rule 7 (P > Pː / V́(R)(s)\_ ; P > B / \_(R)(s)V́) generates systematic fortis/lenis stop alternations conditioned by stress position. It is retained in the Crotonian base ruleset (Appendix A) as documented but **dropped from wékʷos working derivations** pending closer reading of Garnier & Sagot (2017) in full.

## 3. Open architecture

The following remain unspecified:

- **Nominal and verbal morphology:** Stem classes, ending inventory.

## Appendix A: Crotonian — Garnier & Sagot's Rule Set

Twelve rules ordered relative to each other as per Table 1 of Garnier & Sagot (2017). Notation follows Garnier: P = any stop; Pː = fortis counterpart; B = lenis (voiced fricative) counterpart; R = non-syllabic resonant (*r, l, m, n*); H = any laryngeal; Dʰ = any voiced aspirate (*bʰ, dʰ, gʰ, gʷʰ*).

| # | Rule | Environment / Notes |
|---|---|---|
| 1 | Ḱ > K | Centum treatment; merges palatovelar with plain velar |
| 2 | CHC > CăC | Laryngeal between two consonants vocalizes as *ă*; applies to all three laryngeals |
| 3 | bʰR > bR | *bʰ* before non-syllabic resonant loses aspiration before rule 4 fires; bleeds rule 4 for this environment |
| 4 | (#)Dʰ > (#)T | All remaining voiced aspirates → voiceless stops; word-initial explicitly noted; word-internal application masked by rule 7 (fn. 40) |
| 5 | CVRC > CRVC | Metathesis of non-syllabic liquid past preceding vowel |
| 6 | #(s)klV > #(s)kəlV | Schwa epenthesis in word-initial *kl*- and *skl*- clusters |
| 7 | P > Pː / V́(R)(s)\_ and P > B / \_(R)(s)V́ | Verner-like: posttonic stop → fortis; pretonic stop → lenis (voiced fricative); generates systematic fortis/lenis alternations |
| 8 | Barytonesis | Systematic stress retraction to initial syllable; bleeds the conditioning environment for further Verner alternations |
| 9 | VPNV > VNBV | Nasal anticipation with concurrent stop lenition; parallel to Latin *lex unda* but applies to all stops, not dentals only |
| 10 | Cu̯r̥C > CurC | Resyllabification of labiovelar + syllabic resonant + consonant; parallels in Anatolian, Tocharian, Albanian |
| 11 | #pr > #br and #tr > #Tr | Word-initial cluster treatments; labial + *r* → voiced (lenis); dental + *r* → fortis; asymmetric |
| 12 | ow > ū | Monophthongization of back closing diphthong |

### Key structural relationships

- Rule 3 **bleeds** rule 4 for *bʰ*R environments: aspiration stripped before rule 4 can devoice.
- Rule 3 and rules 4+11 **converge** on the same output for word-initial *#bʰr*-: both paths yield *#br*-.
- Rule 7 **masks** the output of rule 4 word-internally (fn. 40).
- Rule 8 **bleeds** further rule 7 alternations by fixing accent initially.

### Diagnostic signature against Ghandwa

| PIE source | Ghandwa outcome | Crotonian outcome | Diagnostic |
|---|---|---|---|
| \*Dʰ (any position) | voiced fricative (β, ð, ɣ) | voiceless stop (T) | strong |

## Bibliography

Garnier, Romain & Benoît Sagot (2017). "A shared substrate between Greek and Italic." *Indogermanische Forschungen* 122(1): 29–60. DOI: 10.1515/if-2017-0002. HAL: hal-01621467.

Related substrate hypotheses for comparison:
- **Temematic** (Holzer 1989): centum + Dʰ > T + T > D; BSl contact zone.
- **Ausonian** (Szemerényi 1991): centum + Dʰ > T; Italic substrate. Functionally a subset of Crotonian with less architecture.
- **Nordwestblock** (Kuhn 1962 and successors): Rhine-Weser area; no reconstructed phonological rule set; useful for suffix morphology and hydronymic root inventory.

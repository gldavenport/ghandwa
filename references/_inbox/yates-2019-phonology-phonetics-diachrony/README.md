# Yates 2019 extraction package

This package contains a clean Markdown extraction of Anthony D. Yates, "The phonology, phonetics, and diachrony of Sturtevant’s Law," *Indo-European Linguistics* 7 (2019), 241–307.

## Files

- `yates-2019-phonology-phonetics-diachrony.md` — main corpus-ready Markdown extraction.
- `yates-2019-phonology-phonetics-diachrony-extraction-report.md` — source assessment, QC notes, glyph map, and further-pass recommendation.

## Note

The extraction is ready for ordinary corpus search. Verify individual forms against the source PDF before quotation or derivational use.

# Etymological Dictionary of the Baltic Inherited Lexicon

## Dictionary — Old Prussian — D

<!-- source-pdf-page: 570; source-page: 556 -->
### dalptan
**OPR** dalptan (EV) 'punch, instrument for punching holes'
**BSL** *dolbt6
**PSL** *dolto n. o (b) 'chisel'
**SL** Ru. dolot6; Cz. dlato; Slk. dlato; PI. dluto; Bulg. dlat6
**IE** OE delfan '<lig'
See →delbti. There are also Slavic forms that continue *delta, e.g. SCr. dlijeto 'chisel'.

<!-- source-pdf-page: 570; source-page: 556 -->
### dirbinsnan
**OPR** dirbinsnan Isg. 'trembling'
This form contains the regular zero grade of a Balto-Slavic root *dreb-, cf. →Lith.
drebinti 'cause to tremble, shake'. l see no need for the generally accepted emendation to dribinsnan. The fact that in East Baltic the analogical zero grade drib- is
common (cf. Lith. →dribinis) does not imply that it was introduced in this Prussian
form as well. Besides, dirb- has been preserved in a number of Latvian forms, e.g.
→Latv. dirbinat 'quiver, walk with small steps, trample, drizzle', dirbet, dirbinet
'tremble'.

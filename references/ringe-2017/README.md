# Ringe 2017 corpus QA-reviewed bundle

This bundle contains the latest generated Markdown/CSV/TSV corpus files for Don Ringe, *From Proto-Indo-European to Proto-Germanic*, second edition (2017), plus a whole-volume QA review.

## Main files

- `front-matter/` — QA-added front matter extraction.
- `chapters/` — chapter Markdown files. Chapter 2, Chapter 3, and Chapter 4 have QA-reviewed versions where needed.
- `index/markdown/` — QA-reviewed Markdown index.
- `index/structured/` — structured index CSV/TSV.
- `references/` — references Markdown and structured CSV/TSV.
- `qa/` — whole-volume QA report, issue list, and metrics table.

## Status

The corpus is suitable for search, comparative-linguistic discovery, and conlang construction planning. Dense paradigms and individual technical forms should be checked against the source PDF before formal citation or before importing into generated lexicons/paradigm tables.

## Notation policy

- PIE / Proto-Core IE / Proto-Central IE use normalized Unicode notation where clear: `kʷ`, `gʷ`, `gʷʰ`, `bʰ`, `dʰ`, `gʰ`, `h₁ h₂ h₃`.
- PGmc and later Germanic forms preserve Ringe-style `kw`, `gw`, and `hw`.
- Attested forms preserve the source spelling/transcription.
- Ambiguous transitional forms should be checked against the PDF.

# Index

## Transcription note

This first-pass index extraction is optimized for comparative-linguistic and conlang-construction search use. The source is a born-digital PDF text layer. Private-use numeral glyphs have been normalized to ordinary Arabic numerals; ligatures have been expanded; PIE index entries have been normalized to stable Unicode notation where practical (`h₁ h₂ h₃`, `ḱ ǵ`, `kʷ gʷ gʷʰ`, `bʰ dʰ gʰ`). Proto-Germanic and later Germanic forms preserve Ringe's Germanic `kw/gw/hw` notation rather than converting it to raised-`w` labiovelar notation.

Large wrapped entries remain line-oriented rather than fully reflowed. For formal citation, check the relevant entry against the PDF.

## I. Protoforms

### A. Proto-Indo-European

the citation when no confusion would result. Post-PIE reconstructions (and
pre-PGmc reconstructions in pre-Grimm’s Law shape) are given in brackets.
Forms of the same lexeme are given in the same entry, usually in alphabetical
order excepting the citation form (if any); for passages in which a whole paradigm is given only one form is indexed, with the notation ‘(parad.)’. Related
lexemes are listed under a single entry unless that would make some too difficult
to find. In a few cases I have listed in a single line related words which are
adjacent in the alphabetical list and appear on exactly the same pages. Suppletive
forms of personal pronouns which are discussed separately are listed separately.

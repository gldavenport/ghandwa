# Extraction report

## Source classification

Source treated as **mixed / scan-derived PDF with OCR text layer**. The PDF metadata identifies Acrobat 5.0 Paper Capture as creator, and the rendered pages are page images with an OCR text layer. The OCR text layer was used as the primary input where coherent. Visual rendering was used only for figure identification/crops and spot checks, not for exhaustive character-level correction.

## Unresolved issues

- `[unclear]` markers inserted: 88. These usually represent blank parenthetical spans where the OCR layer dropped Greek, special characters, or other non-ASCII text.
- The OCR layer drops or corrupts many Greek passages, Indic/Avestan diacritics, underdots, superscript signs, and special transliteration characters. These were **not** fully restored.
- Italics, bold, and small caps are not reliably preserved by the OCR text layer; only the text content was extracted.
- Footnotes are preserved in page order but not fully converted to Markdown footnote syntax.
- Verse, tables, formulas, and parallel-text blocks are preserved mainly by lineation from the text layer; not all were converted into Markdown tables or fenced blocks.
- Figure 3 spans two source pages and is included as two image crops.

# eska-2018-celto-germanic-lexis-second-pass

This package contains a second-pass Markdown extraction of Joseph F. Eska, “Celto-Germanic Lexis in Light of Laryngeal Realism,” from an image-based PDF scan.

## Contents

- `eska-2018-celto-germanic-lexis-second-pass.md` — cleaned Markdown extraction with YAML metadata, page anchors, footnotes, rebuilt diagrams, and normalized bibliography.
- `eska-2018-celto-germanic-lexis-first-pass.md` — the first-pass OCR Markdown retained for comparison.
- `eska-2018-celto-germanic-lexis-raw-ocr.txt` — raw OCR text retained for auditing.

## Second-pass changes

This pass rechecked lexical forms, abbreviations, sound-change diagrams, examples, footnotes, and the bibliography against the page images. The diagrams on the Grimm’s Law and Celtic plosive shift pages were rebuilt as Markdown code blocks, using normalized Unicode phonetic notation.

## Remaining quality notes

This is suitable for corpus/search work and preliminary linguistic reading. It is not a diplomatic transcription. A third pass is only warranted for citation-grade use, a fully diplomatic notation layer, or a separate structured lexical dataset keyed by PGmc./PCelt./PIE forms.

# prosper-inscriptions-todi-vergiate

This package contains a clean, corpus-ready Markdown extraction for:

**The Inscriptions of Todi (Umbria) and Vergiate (Transpadana): A Study in Cisalpine Celtic Epigraphic Habits, Noun Morphology, and the Linguistic Classification of Lepontic**

Files:

- `prosper-inscriptions-todi-vergiate.md` — main Markdown extraction with YAML front matter.
- `prosper-inscriptions-todi-vergiate-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-inscriptions-todi-vergiate-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.
- `prosper-inscriptions-todi-vergiate-excluded-front-matter.txt` — source-volume front matter excluded from the main article Markdown.

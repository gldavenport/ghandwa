# Form Audit Log — Bennett 1969

## Scope

Audited all 76 lexical-form dataset rows against the page-image crops and the corrected v2 Markdown. The audit focused on reconstructed forms, accented vowels, syllabic resonants, superscript labiovelar notation, velar nasal `ŋ`, Gothic/Germanic forms, Celtic comparanda, and citation-sensitive examples.

## Result

No dataset form-string corrections were required after the v2 lexical dataset. The v3 Markdown does include one notation correction outside the dataset rows: source-printed PGmc. `/b/` was restored where v2 had editorially rendered `/ƀ/`.

## Confirmed high-risk forms

- `*wl̥kʷos`, `*wl̥pos`, `*wl̥kʷ-`, `*wl̥p-`: source syllabic-l notation represented as `l̥`.
- `*ljekʷr̥t`: source syllabic-r notation represented as `r̥`.
- `*sekʷnís`: final acute preserved.
- `*bajtá`, `baítē`: accents preserved.
- `*kʷet-wóres`, `-wóres`, `*pet-wóres`: acute on `ó` preserved; no macron added.
- `*xʷeðwōrez`, `*feðwōrez`, `*hvidwōr`, `fidwōr`: long `ō` preserved without added acute.
- `*péŋkʷe`, `*fiŋxʷe`, `*peŋkʷrós`, `*fiŋg(w)raz`: velar nasal `ŋ` preserved.
- `*pimprós`, `*fimbrs`: hypothetical finger p-form path preserved.

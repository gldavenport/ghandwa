# prosper-venetic-names-roman-siscia

This package contains a clean, corpus-ready Markdown extraction for:

**The Venetic Names of Roman Siscia**

Files:

- `prosper-venetic-names-roman-siscia.md` — main Markdown extraction with YAML front matter.
- `prosper-venetic-names-roman-siscia-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-venetic-names-roman-siscia-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

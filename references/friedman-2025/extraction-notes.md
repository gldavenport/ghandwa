## Important cautions

The source is a very large linguistics reference with Greek, Cyrillic, IPA, diacritics, transliteration systems, tables, footnotes, and numbered examples. The embedded text is strong, but not all visual structure can be recovered automatically.

For search, reading, and corpus use, this pass should be sufficient. For publication-quality quotation or a structured linguistic dataset, check the relevant PDF page image for:

- tables with multi-row headers;
- multi-line numbered examples;
- Greek/polytonic Greek and IPA strings;
- footnotes split across PDF page boundaries;
- figure/diagram content.

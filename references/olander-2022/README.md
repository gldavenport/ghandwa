# Olander 2022 Markdown Extraction

This archive contains a clean, corpus-ready Markdown extraction of:

Thomas Olander, ed. *The Indo-European Language Family: A Phylogenetic Perspective*. Cambridge University Press, 2022.

## Files

- `olander-2022-indo-european-language-family.md` — full-volume Markdown extraction.
- `chapters/` — chapter-by-chapter Markdown files plus front matter and index.
- `support/olander-2022-figures-and-tables.md` — figure/table location companion with accessible figure descriptions.
- `support/olander-2022-extraction-notes.md` — method, completed pass notes, limitations, and future-pass guidance.
- `support/olander-2022-character-audit.md` — third-pass character inventory and artifact checklist.

## Pass status

## Extraction notes

This is clean Markdown, not diplomatic transcription. Physical layout, running headers, footers, and Cambridge Core download boilerplate have been removed. Page anchors remain as silent HTML comments for citation and verification.

The extraction is suitable for corpus research. For formal quotation of a specific reconstructed form, table cell, or figure topology, check the PDF page image against the Markdown.

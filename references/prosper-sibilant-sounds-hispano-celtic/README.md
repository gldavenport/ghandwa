# prosper-sibilant-sounds-hispano-celtic

This package contains a clean, corpus-ready Markdown extraction for:

**The Sibilant Sounds of Hispano-Celtic: Phonetics, Phonology and Orthography**

Files:

- `prosper-sibilant-sounds-hispano-celtic.md` — main Markdown extraction with YAML front matter.
- `prosper-sibilant-sounds-hispano-celtic-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-sibilant-sounds-hispano-celtic-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

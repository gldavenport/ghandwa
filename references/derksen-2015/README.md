# Derksen 2015 EDBIL fourth-pass package

This package contains the fourth-pass Markdown extraction and targeted QA outputs for Rick Derksen, *Etymological Dictionary of the Baltic Inherited Lexicon* (Brill, 2015).

## Files

- `derksen-2015-edbil-fourth-pass.md` — cleaned fourth-pass Markdown.
- `derksen-2015-edbil-fourth-pass-notes.md` — summary of pass scope and limitations.
- `derksen-2015-edbil-bibliography-normalized.md` — references section normalized for line wrapping and readability.
- `derksen-2015-edbil-entry-index-fourth-pass.csv` — regenerated entry index.
- `derksen-2015-edbil-fields-fourth-pass.csv` — regenerated field-level extraction.
- `derksen-2015-edbil-reconstructed-forms-fourth-pass.csv` — extracted form-bearing fields with automated flags.
- `derksen-2015-edbil-form-audit.csv` — highest-risk form rows for later page-image verification.
- `derksen-2015-edbil-entry-boundary-audit.csv` — remaining heading anomalies.
- `derksen-2015-edbil-entry-boundary-fixes.csv` — entry-boundary repairs applied in this pass.

## Use

Use the Markdown for reading/searching and the CSVs for database preparation or targeted cleanup. Do not treat OCR-derived reconstructed forms or bibliography characters as citation-grade until checked against the page images.

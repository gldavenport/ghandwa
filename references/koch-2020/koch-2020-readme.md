# Koch 2020 Markdown extraction package

Source: `koch-2020.pdf`

This package contains the second and third passes requested for John T. Koch, *Celto-Germanic: Later Prehistory and Post-Proto-Indo-European vocabulary in the North and West* (2020).

## Contents

- `koch-2020-second-pass.md` — cleaned Markdown extraction.
- `koch-2020-figure-captions.md` — extracted figure-caption companion file.
- `koch-2020-lexical-dataset.md` — readable structured lexical entries from §§39–51.
- `koch-2020-lexical-dataset.csv` — tabular lexical dataset.
- `koch-2020-lexical-dataset.json` — JSON lexical dataset.
- `koch-2020-bibliography-normalized.md` — normalized bibliography and abbreviation list.
- `koch-2020-bibliography.tsv` — bibliography table.
- `koch-2020-second-third-pass-qa.md` — quality report and caveats.
- `koch-2020-first-pass.md` — previous first-pass Markdown retained for comparison.
- `koch-2020-extraction-notes.md` — previous first-pass notes retained for context.

## Notes

The third-pass lexical dataset is machine-assisted. It is useful for search, filtering, and downstream analysis, but not a final philological edition. Entries with `quality_flags` should be reviewed first if used for citation, linguistic argument, or conlang derivational work.

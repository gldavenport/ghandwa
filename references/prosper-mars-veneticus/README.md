# prosper-mars-veneticus

This package contains a clean, corpus-ready Markdown extraction for:

**Mars Veneticus and the “Palma Rule”**

Files:

- `prosper-mars-veneticus.md` — main Markdown extraction with YAML front matter.

- `prosper-mars-veneticus-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

# Kroonen 2012 second and third passes

Source: `kroonen-2012.pdf`

## Included files

- `kroonen-2012-non-indo-european-root-nouns-germanic-pass2.md` — revised clean Markdown with targeted character/form verification corrections.
- `kroonen-2012-root-nouns-lexical-dataset.md` — readable structured lexical dataset.
- `kroonen-2012-root-nouns-lexical-dataset.csv` — CSV version of the lexical dataset.
- `kroonen-2012-bibliography-normalized.md` — source-preserving normalized bibliography.

## Second-pass scope

The second pass checked section structure, footnote numbering, the six primary root-noun candidates, major reconstructed forms, Greek forms central to the argument, and bibliography line-break artifacts against the PDF text layer and selected rendered pages. The PDF is text-based; the main risks were character fidelity, line-break joins, and scholarly notation, not OCR.

Notable corrections include: adding a missing space in `PIE *u̯ became *b`; normalizing laryngeal numerals to Unicode subscripts (`h₁`, `h₂`); correcting `Haas et al. 2010` to `Haak et al. 2010`; correcting `Causasian` to `Caucasian`; joining `Near East-ern`; and normalizing the dotless-i nasalized suffix form to `*ĩd(h)-`.

## Third-pass scope

The third pass produced a structured lexical dataset for the six main root-noun cases and selected comparanda used in the article’s substrate argument, plus a lightly normalized bibliography.

## Further-pass recommendation

No general fourth pass is needed for corpus use. A future specialist pass would only be useful for citation-grade philological fidelity: checking every Greek accent/breathing and every reconstructed form against page images at high zoom.

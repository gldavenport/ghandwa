# prosper-inscription-cabeco-das-fraguas

This package contains a clean, corpus-ready Markdown extraction for:

**The Inscription of Cabeço das Fráguas Revisited: Lusitanian and Alteuropäisch Populations in the West of the Iberian Peninsula**

Files:

- `prosper-inscription-cabeco-das-fraguas.md` — main Markdown extraction with YAML front matter.
- `prosper-inscription-cabeco-das-fraguas-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-inscription-cabeco-das-fraguas-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

# West 2007 Markdown Extraction — Additional Passes

## Source

M. L. West, *Indo-European Poetry and Myth* (Oxford University Press, 2007).

## Main file

- `west-2007.md` — third-pass clean Markdown extraction.

## Companion files

- `west-2007-outline.md` — structural outline from the PDF outline/book table of contents.
- `west-2007-extraction-notes.md` — pass log, character-audit notes, known limitations, and remaining artifact counts.

## What changed from first pass

The first pass was a clean structural extraction. These additional passes targeted structure, duplicate headings, hyphenation, common ligature-spacing artifacts, and high-frequency character corruption in Greek, Sanskrit/Vedic transliteration, Avestan/Indic diacritics, and PIE notation.

## Use guidance

Use `west-2007.md` as the current corpus copy. Verify local page images before quoting specialized forms, especially Greek, Avestan, Vedic, and PIE reconstructions.

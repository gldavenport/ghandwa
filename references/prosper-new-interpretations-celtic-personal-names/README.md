# prosper-new-interpretations-celtic-personal-names

This package contains a clean, corpus-ready Markdown extraction for:

**Sifting the Evidence: New Interpretations on Celtic and Non-Celtic Personal Names of Western Hispania in the Light of Phonetics, Composition and Suffixation**

Files:

- `prosper-new-interpretations-celtic-personal-names.md` — main Markdown extraction with YAML front matter.
- `prosper-new-interpretations-celtic-personal-names-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-new-interpretations-celtic-personal-names-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

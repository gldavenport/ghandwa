# prosper-ie-personal-names-pannonia-northern-italy

This package contains a clean, corpus-ready Markdown extraction for:

**The Indo-European Personal Names of Pannonia, Noricum and Northern Italy: Comparative and Superlative Forms in Celtic, Venetic, and South-Picene**

Files:

- `prosper-ie-personal-names-pannonia-northern-italy.md` — main Markdown extraction with YAML front matter.
- `prosper-ie-personal-names-pannonia-northern-italy-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-ie-personal-names-pannonia-northern-italy-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

# prosper-language-change-crossroads

This package contains a clean, corpus-ready Markdown extraction for:

**Language Change at the Crossroads: What Celtic, What Venetic, and What Else in the Personal Names of Emona?**

Files:

- `prosper-language-change-crossroads.md` — main Markdown extraction with YAML front matter.
- `prosper-language-change-crossroads-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-language-change-crossroads-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

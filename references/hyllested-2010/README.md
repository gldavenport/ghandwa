# hyllested-2010-precursors-celtic-germanic

This folder contains a second-pass Markdown extraction of Adam Hyllested, "The Precursors of Celtic and Germanic" (2010), from the uploaded image-only PDF.

## Files

- `hyllested-2010-precursors-celtic-germanic.md` — cleaned Markdown extraction with YAML front matter, structural headings, numbered lexical entries, footnotes, and references.

## Extraction notes

The source PDF is image-only and contains 22 scanned pages. The first pass established the article structure and cleaned the OCR into readable Markdown. The second pass checked the rendered page images and page-level OCR against the Markdown and focused on corpus-readiness rather than a fully diplomatic edition.

Second-pass checks completed:

- verified the article has 96 numbered lexical entries;
- verified all footnotes 1-25 are present;
- verified the major section sequence, including section 5 and the conclusions;
- corrected a set of visible OCR/normalization problems in reconstructed forms, language labels, and Fennic/Celtic/Germanic forms;
- updated front matter to mark this as a second-pass extraction.

Remaining caution areas:

- Greek forms and some non-Latin-script material are still normalized rather than fully diplomatic;
- a few reconstructed forms with laryngeals, superscripts, and uncertain OCR may still require scan-level verification for citation use;
- bibliography formatting is cleaned for corpus use, not fully typographic.

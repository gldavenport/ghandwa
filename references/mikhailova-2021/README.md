# Mikhailova 2021 Markdown extraction package

This package contains a clean, corpus-ready Markdown extraction of:

Tatyana A. Mikhailova, “Night-mare: on the origin of a trope in Celtic and Germanic (a response to Stephen Pax Leonard),” *Journal of Language Relationship / Вопросы языкового родства* 19/1 (2021): 15–24.

## Contents

- `mikhailova-2021-night-mare-origin-trope.md`  
  Main Markdown extraction with YAML front matter, source page anchors, reconstructed section structure, footnote, references, and Russian abstract.

- `mikhailova-2021-extracted-forms.md`  
  Selected extracted forms and etyma table for quick corpus lookup.

- `mikhailova-2021-extraction-notes.md`  
  Pass status, cleanup notes, character-audit notes, and remaining uncertainty.

## Extraction approach

The extraction is clean Markdown rather than diplomatic transcription. Physical layout, running heads, footers, and ordinary page numbers were removed. Logical structure, citations, quotations, footnote content, references, source page anchors, and special linguistic notation were preserved.

The extraction has been through first-pass extraction, second-pass structural cleanup, and third-pass special-character/linguistic-form review.

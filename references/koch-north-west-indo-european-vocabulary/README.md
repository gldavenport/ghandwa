# Koch 2024 extraction package

This folder contains a Markdown extraction of John T. Koch, “Celto-Germanic and North-West Indo-European vocabulary: Resonances in myth and rock art iconography,” chapter 10 of *Indo-European Interfaces: Integrating Linguistics, Mythology and Archaeology*.

## Files

- `koch-2024-celto-germanic-north-west-indo-european-vocabulary.md` — cleaned Markdown extraction.
- `images/figure-1.jpg` through `images/figure-7.jpg` — figures copied from the EPUB and linked from the Markdown.

## Extraction notes

- Primary text source: `indo-european-interfaces.epub`, chapter file `Ops/018.xhtml`.
- Control source: chapter PDF, rendered to verify page count and figure coverage.
- Footnote references were converted to Markdown footnotes.
- EPUB image filenames were normalized to `figure-1.jpg` through `figure-7.jpg`.
- Inline italics, bold, superscripts, and subscripts are preserved using Markdown/HTML-compatible inline markup.

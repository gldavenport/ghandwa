# prosper-celtic-venetic-contact

This package contains a clean, corpus-ready Markdown extraction for:

**Celtic and Venetic in Contact: The Dialectal Attribution of the Personal Names in the Venetic Record**

Files:

- `prosper-celtic-venetic-contact.md` — main Markdown extraction with YAML front matter.
- `prosper-celtic-venetic-contact-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-celtic-venetic-contact-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

# Ringe 2024 remaining uncertain forms

No automatic pass can fully guarantee every dense linguistic form in a 415-page work of this kind. After the targeted visual-text verification pass, the highest-value remaining manual checks would be:

1. Dense paradigms in section 2.3.3, where many reconstructed forms appear without asterisks because the author explicitly omits them for typographical clarity.
2. Greek forms in sections 3-7 where the PDF text layer mixes Greek characters with Latin `h`, `j`, `w`, and occasionally Latin `o` for omicron.
3. Footnotes containing compact etymological argumentation, especially those with Sanskrit, Avestan, Hittite, Tocharian, Old Irish, or Gothic forms.
4. Bibliography entries with non-English names and titles, where diacritics and Greek titles may still need source-image checking.
5. Index entries, because index typography is compressed and more likely to produce token-boundary noise.

The main Markdown now follows the visual glyph positions for superscript/subscript notation. Future correction should be done by checking the page image, not by normalizing forms to external conventions.

# Koch 2019 extraction handoff

## Current state

The Markdown file has received a second cleanup pass after the initial extraction. It is suitable for reading and corpus search. The pass focused on clean prose rather than a diplomatic reproduction of the PDF layout.

## What was cleaned in the second pass

- Repaired obvious small-caps/OCR capitalization artifacts in the body and references, especially author names and sentence starts.
- Cleaned several broken hyphenation and spacing artifacts in the references and URLs.
- Checked and regularized the major phonological tables in Working Hypothesis 6.
- Repaired key linguistic notation where the PDF extraction displaced characters, especially the Derwent example: `*Dreu̯entiH₂-` and `*Deru-u̯ent-`.
- Converted the major cited prose excerpts into Markdown block quotes for readability.
- Moved future-pass notes out of the extracted Markdown and into this handoff file.

## Possible future pass

A future, higher-effort pass would be useful only if this text is going to be used as a citation-sensitive reference rather than as a corpus/research aid. That pass should:

1. Verify every reference entry against the source PDF or published bibliographic records, especially very long multi-author archaeogenetics entries.
2. Check all author-name particles and diacritics in references 26, 30, 43, 83, 107, and 112.
3. Verify exact source-title capitalization in the bibliography; this pass normalized obvious artifacts but did not attempt journal-style bibliographic standardization.
4. Compare all PIE forms and phonological symbols against the PDF at high zoom, especially forms with laryngeals, combining diacritics, and labiovelars.
5. Replace accessible figure descriptions with fuller descriptions if map-label content becomes important for later archaeological/geographic analysis.
6. Decide whether citation markers should remain as plain inline numbers, become bracketed references, or be converted into footnotes/endnotes for a larger standardized archive.

## Known limitations remaining

- The PDF is a two-up layout. The extraction intentionally ignores page layout and preserves structure as readable Markdown.
- Figures are represented by captions and brief accessible descriptions, not by embedded images.
- Reference-list cleanup was targeted, not a full bibliographic audit.
- Some capitalization in titles may reflect the cleanup pass rather than exact source typography.

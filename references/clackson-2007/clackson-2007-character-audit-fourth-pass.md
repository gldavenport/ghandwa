# clackson-2007 fourth-pass character audit

## Pass purpose

This pass focused on character fidelity rather than broad structural rewriting. It resolved the prior bracketed uncertainty markers where the PDF text layer, PyPDF glyph names, and selected page images supported a likely reading.

## Main changes

- Restored syllabic-resonant notation using combining ring below, e.g. `r̥`, `l̥`, `m̥`, `n̥`, and long syllabic forms such as `r̥̄`.
- Restored common nasal-vowel notation such as `ą`, `ę`, and `ǫ` where the PDF glyph names indicated a subscript half-ring/ogonek-like mark.
- Restored several Greek bibliography/title items: `κλέος ἄφθιτον`, `ἕσπερος`, and `τε épique`.
- Restored `Kuryłowicz`, `astł`, `gl̥ō-`, `vŕ̥ka-`, `ŕ̥kṣa-`, `śr̥ṇóti`, and related index/table forms where source checks supported the reading.
- Conservatively normalized high-confidence extracted sequences such as dotless-i combinations (`´ı` → `í`), caron sequences (`sˇ` → `š`), and common vowel macron/acute sequences.

## Remaining uncertainty markers

- Remaining `[unclear...]` markers: 0

## Limits of this pass

Some dense comparative tables and index pages still preserve line-oriented extraction artifacts from previous passes. This pass improves character fidelity inside those artifacts, but it does not fully rebuild every table into a publication-grade table. For ordinary corpus research, the character state is materially improved; for exact citation of a table or index entry, check the source PDF page anchor.

# Kroonen 2016 extraction package

Source: `kroonen-2016.pdf`

Article: Guus Kroonen, “On the Origin of Greek μῆλον, Latin mālum, Albanian mollë and Hittite šam(a)lu- ‘apple’,” *The Journal of Indo-European Studies* 44.1/2 (Spring/Summer 2016), 85-91.

## Files

- `kroonen-2016-greek-melon-latin-malum-third-pass.md` — third-pass clean Markdown extraction.
- `kroonen-2016-lexical-comparanda.md` — structured Markdown table of forms and comparanda.
- `kroonen-2016-lexical-comparanda.csv` — same dataset as CSV.
- `kroonen-2016-references-normalized.md` — normalized bibliography.
- `kroonen-2016-references-normalized.json` — structured bibliography data.
- `kroonen-2016-extraction-notes.md` — pass notes and remaining uncertainties.

## Pass status

This is a third-pass extraction suitable for clean Markdown corpus use. The main remaining source-internal issue is the `Tournier` / `Tourner` mismatch, preserved and documented.

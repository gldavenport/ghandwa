# prosper-sabellic-accusative-plural-endings

This package contains a clean, corpus-ready Markdown extraction for:

**The Sabellic Accusative Plural Endings and the Outcome of the Indo-European Sibilants in Italic**

Files:

- `prosper-sabellic-accusative-plural-endings.md` — main Markdown extraction with YAML front matter.
- `prosper-sabellic-accusative-plural-endings-forms.tsv` — machine-extracted technical/epigraphic/reconstructed-form list for search and review.
- `prosper-sabellic-accusative-plural-endings-extraction-notes.md` — pass notes, cautions, and further-pass recommendation.

# kroonen-2022 third-pass Markdown package

This package contains the third-pass Markdown extraction of Kroonen et al. 2022.

## Main files

- `kroonen-2022-cereal-terminology.md` — third-pass Markdown extraction.
- `third-pass-verification-report.md` — notes on what the third pass verified and remaining limitations.

## Pass status

This is suitable for close reading, corpus search, and form-level checking. Another pass is not recommended unless the goal becomes full bibliographic authority control or a manually checked diplomatic edition.

# Bjørn 2017 Markdown extraction package

Source: Rasmus Gudmundsen Bjørn, *Foreign elements in the Proto-Indo-European vocabulary: A Comparative Loanword Study* (2017).

This is the fourth-pass package. It supersedes the previous package for corpus use.

## Files

- `bjorn-2017-foreign-elements-proto-indo-european-vocabulary.md` — main clean Markdown extraction.
- `bjorn-2017-lexical-dataset.csv` — structured lexical dataset extracted from the wordlist.
- `bjorn-2017-lexical-dataset.md` — readable Markdown version of the lexical dataset.
- `bjorn-2017-bibliography-extracted.md` — bibliography as extracted from the source.
- `bjorn-2017-bibliography-normalized.md` — lightly normalized one-reference-per-bullet companion bibliography.
- `bjorn-2017-extraction-notes.md` — fourth-pass notes, character audit, and validation results.

## Pass status

Fourth pass complete. The main improvements are character cleanup in reconstructed forms, reduced whitespace/word-join artifacts, lexical item validation, and a bibliography normalization companion.

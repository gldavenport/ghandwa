## ésti "to be" — Present Active Indicative

PIE root: \*h₁es- "to be"

|  | Singular | Plural |
|---|---|---|
| 1st | *ézmi* | *ezmós* |
| 2nd | *ézi* | *esté* |
| 3rd | *ésti* | *sénti* |

### Derivation notes

- **1sg** *ézmi*: \*h₁ésmi → ésmi (H-B4 drops h₁); s→z between voiced sounds (é_m) → ézmi
- **2sg** *ézi*: \*h₁ési → ési; s→z between voiced sounds (é_i) → ézi
- **3sg** *ésti*: \*h₁ésti → ésti; s before voiceless t, no voicing. Already in TSV.
- **1pl** *ezmós*: PIE \*h₁smós → smós (H-B4); bare sm- onset universally repaired by analogy in PIt, PC, PGmc independently → analogical e- adopted for Ghandwa → esmós; s→z between voiced sounds (e_m) → ezmós
- **2pl** *esté*: PIE \*h₁sté → sté (H-B4); bare st- onset universally repaired by analogy → analogical e- adopted → esté; s before voiceless t, no voicing
- **3pl** *sénti*: \*h₁s-énti → sénti (H-B4 drops h₁; s word-initial before vowel, no voicing trigger). Already in TSV. Now the paradigmatic outlier — zero-grade root with no analogical e-, while 1pl and 2pl restored it. Daughters may level this differently.

### Comparative basis for analogical forms

PIE 1pl \*h₁smós and 2pl \*h₁sté both have zero-grade root (no \*e). Word-initial h₁ drops, leaving bare cluster onsets sm-, st-. All three western sisters repaired independently:

- **PIt**: added e- (esomos, estes)
- **PC**: added e- (esomosi, estesi)
- **PGmc**: added i- (izum, izud — from earlier esom?, esde?)

Universal independent repair → Ghandwa adopts analogical e- in 1pl, 2pl. 3pl *sénti* not repaired because it already has a vowel after the s (s-énti), so the onset was never problematic.

## βā́ti "to speak (authoritatively)" — Present Active Indicative

PIE root: \*bʰeh₂- "to speak"
Register: sacred, prophetic, legal. Contrasts with gvéteti (everyday).

|  | Singular | Plural |
|---|---|---|
| 1st | *βā́mi* | *βamós* |
| 2nd | *βā́zi* | *βaté* |
| 3rd | *βā́ti* | *βánti* |

### Derivation notes

**Singular** — full-grade root \*bʰéh₂-:
- **1sg** *βā́mi*: \*bʰéh₂mi → H-A colors é→á; H-B2 á before h₂ before C → ā́, h₂ drops; bʰ→β → βā́mi
- **2sg** *βā́zi*: \*bʰéh₂si → same laryngeal path → bʰā́si; s→z between voiced sounds (ā́_i) → βā́zi
- **3sg** *βā́ti*: \*bʰéh₂ti → same laryngeal path → bʰā́ti; bʰ→β → βā́ti

**Plural** — zero-grade root \*bʰh₂-:
- **1pl** *βamós*: \*bʰh₂mós → H-B5 (h₂ between consonants bʰ_m, initial syllable → a) → bʰamós; bʰ→β → βamós
- **2pl** *βaté*: \*bʰh₂té → H-B5 (h₂ between consonants bʰ_t → a) → bʰaté; bʰ→β → βaté
- **3pl** *βánti*: \*bʰh₂énti → H-A (h₂ colors following é→á); H-B3 (h₂ adjacent to vowel → drops) → bʰánti; bʰ→β → βánti

### Paradigm notes

The sg/pl alternation is ā ~ a throughout — long vowel in the singular from laryngeal coloring + compensatory lengthening, short a in the plural from H-B5 (1pl, 2pl) or H-A coloring (3pl). No bare-cluster onsets, so no analogical repairs needed (unlike ésti). The pattern parallels ðḗti ~ ðánti (ē ~ a from \*dʰeh₁-), making the two laryngeal-root athematics a natural paradigmatic class.

## gvéteti "to say, tell" — Present Active Indicative

PIE root: \*gʷet- "to say"
Register: everyday speech, narrative, reported speech. Contrasts with βā́ti (sacred).

|  | Singular | Plural |
|---|---|---|
| 1st | *gvétō* | *gvétomos* |
| 2nd | *gvétezi* | *gvétete* |
| 3rd | *gvéteti* | *gvétonti* |

### Derivation notes

Thematic root present — no root ablaut, no laryngeals. Thematic vowel alternates e (before consonant-initial endings) / o (before nasal-initial endings and 1sg).

- **1sg** *gvétō*: \*gʷét-ō (< \*-o-h₂; thematic vowel + laryngeal → long ō) → gvétō
- **2sg** *gvétezi*: \*gʷét-e-si → gvétesi; s→z between voiced sounds (e_i) → gvétezi
- **3sg** *gvéteti*: \*gʷét-e-ti → gvéteti. No voicing triggers.
- **1pl** *gvétomos*: \*gʷét-o-mos → gvétomos
- **2pl** *gvétete*: \*gʷét-e-te → gvétete
- **3pl** *gvétonti*: \*gʷét-o-nti → gvétonti

### Paradigm notes

The thematic present is the most transparent formation in Ghandwa — the root never changes, the thematic vowel is predictable, and the only phonological process is s→z voicing in 2sg. This paradigm serves as the baseline model for all thematic root presents (édeti, véideti, sékveti, gvémeti, tekseti, kléveti).

## pípoti "to drink" — Present Active Indicative

PIE root: \*peh₃- "to drink"
Present stem: \*pi-ph₃- (reduplicated thematic; zero-grade root throughout)

|  | Singular | Plural |
|---|---|---|
| 1st | *pípō* | *pípomos* |
| 2nd | *pípozi* | *pípote* |
| 3rd | *pípoti* | *píponti* |

### Derivation notes

The stem is \*pi-pH₃-, with the thematic vowel following h₃. In every person, h₃ colors the adjacent thematic vowel (e→o, o stays o) then drops via H-B3. This collapses the usual thematic e/o alternation — the thematic vowel surfaces as **o throughout**, confirmed by Latin *bibō* and Sanskrit *píbati* both showing o-grade.

- **1sg** *pípō*: \*pi-ph₃-o-h₂ → h₃ adjacent to o (no change), drops; o+h₂ → VH→V̄ → ō → pípō
- **2sg** *pípozi*: \*pi-ph₃-e-si → h₃ colors e→o, drops → píposi; s→z between voiced sounds (o_i) → pípozi
- **3sg** *pípoti*: \*pi-ph₃-e-ti → h₃ colors e→o, drops → pípoti
- **1pl** *pípomos*: \*pi-ph₃-o-mos → h₃ adjacent to o, drops → pípomos
- **2pl** *pípote*: \*pi-ph₃-e-te → h₃ colors e→o, drops → pípote
- **3pl** *píponti*: \*pi-ph₃-o-nti → h₃ adjacent to o, drops → píponti

### Paradigm notes

The accent stays on the reduplication syllable throughout (píp-), matching Sanskrit *píbati*. The root never changes shape — no ablaut, no consonant alternation. The only interesting feature is the collapsed thematic vowel: always o, never e, because h₃ colors every instance. This makes pípoti look superficially like a thematic present with an anomalous o-only vowel — the reduplication and the vowel collapse are the markers that set this class apart.

The p...p sequence at the morpheme boundary (pí**p**o**t**i) is the same cluster that Latin voiced to b...b (*bibō*) and Celtic deleted (*ibid*). Ghandwa preserves it unchanged — but see Pending Tasks re h₃ voicing rule.

## ðéɣveti "to burn" — Present Active Indicative

PIE root: \*dʰegʷʰ- "to burn"

|  | Singular | Plural |
|---|---|---|
| 1st | *ðéɣvō* | *ðéɣvomos* |
| 2nd | *ðéɣvezi* | *ðéɣvete* |
| 3rd | *ðéɣveti* | *ðéɣvonti* |

### Derivation notes

Standard thematic root present. No laryngeals, no ablaut complications. dʰ→ð, gʷʰ→ɣʷ (→ orthographic ɣv) in Stage 2. s→z in 2sg (e_i). Identical formation to gvéteti.

### Present active participle

\*dʰegʷʰ-ó-nt-s → ðeɣvónts (nom.sg.m.), gen. ðeɣvantés.

Epithet candidates for Agnis:
- *ðeɣvóntios* — "the blazing one" (solemn/hymnic)
- *ðeɣvontiskos* — "the little blazing one" (folk/affectionate, Baby Agnis)

## ðoɣvéieti "to kindle, ignite, set on fire" — Present Active Indicative

PIE root: \*dʰogʷʰ-éye- (éye-causative of \*dʰegʷʰ-)
First éye-causative in the lexicon.

|  | Singular | Plural |
|---|---|---|
| 1st | *ðoɣvéiō* | *ðoɣvéiomos* |
| 2nd | *ðoɣvéiezi* | *ðoɣvéiete* |
| 3rd | *ðoɣvéieti* | *ðoɣvéienti* |

### Derivation notes

The éye-causative has o-grade root + -éye- suffix. dʰ→ð, gʷʰ→ɣʷ in Stage 2; y→i orthographically. The suffix -éie- carries the accent throughout. Thematic vowel follows the suffix: -éie-ti, -éie-nti (the 3pl ending is -nti directly after the suffix, not -onti, because éye-presents use athematic-style endings on the suffix).

s→z in 2sg (e_i). Otherwise no phonological processes fire.

### Semantic note

ðéɣveti = "burns" (inherent state or general action). ðoɣvéieti = "kindles, ignites" (deliberate causation). The basic verb is what Agnis does; the causative is what humans do to invoke Agnis — the priest kindles, the smith kindles, the worshipper kindles. Agent noun for "fire-starter" epithet: pending.
